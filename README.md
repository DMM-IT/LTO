# LTO MIMAROPA Queue Management System

A real-time queue management system for LTO district offices in MIMAROPA region.

## Live Site
- Home: https://lto-pink.vercel.app
- Login: https://lto-pink.vercel.app/login.html
- Kiosk: https://lto-pink.vercel.app/kiosk.html
- Queue Board: https://lto-pink.vercel.app/board.html
- Counter: https://lto-pink.vercel.app/counter.html
- Admin Panel: https://lto-pink.vercel.app/admin.html

## How to Update the Code
1. Open GitHub Codespaces: https://github.com/codespaces
2. Make your changes
3. Run these commands:
   cd /workspaces/LTO/lto-queue-system
   git add .
   git commit -m "describe your changes"
   git push

## Tech Stack
- Frontend: HTML, CSS, JavaScript (hosted on Vercel)
- Backend: Node.js, Express, Socket.io (hosted on Railway)
- Repository: GitHub

## Troubleshooting
- Site not loading: Check vercel.com/dashboard and redeploy
- Connection error: Check railway.app, server may be sleeping
- Changes not showing: Wait 1-2 minutes for Vercel to redeploy
